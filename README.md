O CRUD Petrobras é um sistema completo de gerenciamento de eventos, desenvolvido com arquitetura separada entre frontend e backend, seguindo boas práticas de desenvolvimento moderno.

O objetivo do projeto é permitir o gerenciamento de eventos de forma simples, eficiente e escalável, com comunicação via API REST, garantindo integração entre sistemas e facilidade de manutenção.

⚙️ Tecnologias Utilizadas
🖥 Backend (API REST)
Java 17+
Spring Boot
Spring Web (REST APIs)
Spring Data JPA
Hibernate
Flyway (versionamento e controle do banco de dados)
MySQL
Maven
🌐 Frontend
Angular 21
TypeScript
HTML5 / CSS3
Angular HTTP Client (consumo da API REST)
🧰 Ferramentas e Integrações
Git & GitHub
Swagger (documentação interativa da API)
Postman (testes e validação de endpoints)


Swagger (Documentação da API)

A API é documentada automaticamente com Swagger, permitindo:

Visualizar todos os endpoints
Testar requisições diretamente no navegador
Entender os modelos de dados (DTOs)
Facilitar integração com outros sistemas

🔗 Exemplo de acesso:   http://localhost:8080/swagger-ui.html


Objetivo do Projeto
Criar uma aplicação full-stack moderna
